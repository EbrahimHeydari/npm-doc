// semantic version

"react": "17.0.2"

17: major version => تغییرات اساسی

0:  minor version => افزودن قابیلت های جدید

2:  patch version => رفع مشکلات کوچک


"react": "*" => آپدیت تا آخرین نسخه موجود ( از این مورد نباید استفاده شود) 

"react": "^17.0.2" => آپدیت تا major version 

"react": "~17.0.2" => آپدیت تا minor version 

"react": "17.0.2" => نصب یا آپدیت تا همان ورژن


upgrade your app:

npm version patch: ارتقا نرم افزار به نسخه patch بالاتر
"17.0.2" => "17.0.3"

npm version minor: ارتقا نرم افزار به نسخه minor بالاتر
"17.0.2" => "17.3.0"

npm version major: ارتقا نرم افزار به نسخه major بالاتر
"17.0.2" => "18.0.0"

نکته: برای اینکه نرم افزار را بتوانید ارتقا دهید، باید نسخه جدید در فایل گیت پوشه ثبت (کامیت) شده باشد

npm publish: انتشار نسخه جدید
